### 2. 实验 / 项目

**Venus 复现（Aesthetic Guidance and Cropping）：**

- 具体工作内容：完成 Venus 论文复现的环境搭建与数据准备工作
- 原理 / 方法：Venus 采用两阶段训练策略——Stage1 通过渐进美学问答赋予大模型美学指导能力，Stage2 通过思维链裁剪推理激活美学裁剪能力；复现需准备 AesGuide 基准数据、FLMS 基准数据及对应训练数据集
- 结果与数据分析：(1) 完成环境搭建；(2) 下载数据；(3) 完成预训练权重下载：Venus-Q-Stage1 和 Venus-Q-Stage2 
- 存在的问题与改进思路：训练阶段尚未启动，需要 GPU 资源；下一步完成 API 基线评测后启动实际训练

**PhotoAgent 复现（Agentic Photo Editing）：**

- 具体工作内容：完成 PhotoAgent 论文复现的核心代码架构搭建
- 原理 / 方法：PhotoAgent 将照片编辑建模为闭环决策过程——Perceiver 提出候选动作、Planner 基于 MCTS 搜索最优动作序列、Executor 执行编辑、Evaluator 提供视觉反馈形成闭环；复现需实现四模块协同的 Agent 管线
- 结果与数据分析：(1) 完成核心模块实现；(2) 完成配置系统搭建，支持国内 API 代理；(3) 代码完成度约 95%
- 存在的问题与改进思路：识别出 3 个关键 Bug——(1) best_editor 未赋值导致记忆系统无法学习；(2) 图像尺寸检查拒绝有效语义编辑（Flux/GPT 输出 1024×1024 被误判）；(3) MCTS 可能创建重复节点。需修复后进行完整推理测试

**AesFormer 复现准备（Aesthetic Photo Reconstruction）：**

- 具体工作内容：完成 AesFormer 论文复现的脚本配置与硬件评估
- 原理 / 方法：AesFormer 采用两阶段框架——AesThinker（七维渐进美学分析 + GRPO-A 强化学习）和 AesEditor动作条件编辑；复现涉及 Stage1aSFT 冷启动、Stage1bGRPO-A 强化训练、Stage2LoRA 编辑训练

### 3. 其他进展

**计算美学综述拓展阅读（配合论文 [1] 绘画艺术图像的计算美学）：**

为深入理解计算美学领域的发展脉络，阅读了以下三篇拓展资料：

(1) 知乎专栏 | 鲍晓杰. 图像美学质量评价调研报告. 2019.

系统梳理了图像美学质量评价的研究现状，涵盖：(a) 美学数据库--Photo.Net；(b) 特征提取从手工设计到深度特征的演进，指出 CNN 固定输入尺寸对美学评估的特殊挑战——裁剪/缩放/填充会损害原始美感；(c) 代表性方法：RAPID--双列 CNN 融合全局与局部信息、DMA-Net--多 patch 统计聚合、A-Lamp--自适应 patch 选择+属性图布局感知、NIMA--Google，输出 1-10 分直方图分布，Earth Mover Distance 损失函数等。该综述明确了"特征提取+决策"的主流研究范式，以及深度学习方法相比手工特征的显著优势。

(2) 知乎专栏. 计算机美学综述--涵盖美学评分、图片美化、自动裁剪、自动构图、美学评价五大方向. 2019.

从更宏观的视角梳理了计算美学的五大研究方向：(a) 美学评分：从 RAPID--ACM MM 2014，最早深度学习尝试到 NIMA--Google，直方图分布预测，核心挑战在于美的定义模糊、评分主观性强，依赖 AVA/CUHK-PQ 等大规模众包数据集；(b) 图片美化：Aesthetic-Driven Enhancement--GAN 弱监督，利用已训练的判别器引导生成器和 Google Creatism--将增强分解为互不影响的滤波器链，针对饱和度/光照等分别训练；(c) 自动裁剪：从显著性检测到 A2-RL--强化学习，CVPR 2018再到 GAIC--CVPR 2019，提出 SRCC 和 R@K 评价指标替代 IoU；(d) 自动构图：用 Siamese Network 学习专业照片与随机裁剪的构图差异，以及基于知识蒸馏的快速窗口搜索；(e) 美学评价：从 ICCV 2017 的 PCCD 数据集--7 维度描述+评分到 ACM MM 2019 的 15 万+大规模属性评价数据集。该综述指出计算美学的核心难点在于"如何定义问题"和"如何建模主观审美"，几乎所有方法都是针对数据集构建的。

(3) VALSE Webinar 2020 | 李雷达, 马柯德. 图像质量评价：感知建模VS机器学习. BV1gQ4y1P72m.

分别从感知建模和机器学习两个角度讨论图像质量评价：(a) 个性化图像美学评价——如何建模不同人群的审美偏好差异；(b) 结构相似性与纹理相似性的统一——图像质量评估中 SSIM 等客观指标与主观感知的对齐问题。厘清了"计算机美学 "与"图像质量评估 "两个相关但不同课题的边界。
