# Research Weekly Reports

科研周记仓库，用于记录每周研究进展、论文阅读、实验结果与下一步计划。

## 目录结构

```
research-weekly-reports/
├── README.md               # 项目总说明
├── weekly/                 # 所有周记按周放这里
│   ├── 20260518-0524.md
│   ├── 20260525-0531.md
│   └── ...
├── assets/                 # 配图、截图、笔记图
├── docs/                   # 论文笔记、调研笔记（可选）
│   └── papers/             # 读过的领域相关论文 PDF
└── .gitignore
```

## 周记命名规范

- 文件名格式：`YYYYMMDD-YYYYMMDD.md`（起止日期）
- 示例：`20260518-0524.md` 表示 2026 年 5 月 18 日至 5 月 24 日的周记

## 资源管理

- `assets/` — 存放与周记相关的图片、截图等资源，命名格式：`YYYYMMDD-描述.png`
- `docs/` — 存放较长的论文笔记、调研文档等独立文档
